# 🐾 PetVerse

> AI-Powered Pet Care & Management System

PetVerse is a web-based platform designed to simplify pet care by combining pet management, AI-powered assistance, online shopping, and veterinary services in one place.

---

## 🚀 Features

- 🔐 Secure User Authentication (Login & Signup)
- 🐶 Pet Profile Management
- 🤖 AI Breed Detection from Images
- 💬 AI Chat Assistant for Pet Queries
- 📖 Pet Care Guide
- 🛒 Pet Products Shopping
- 🩺 Veterinary Appointment Booking
- 🏡 Pet Adoption
- 👤 User Dashboard & Profile
